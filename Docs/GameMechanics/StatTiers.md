| Tier | Modifier | Description |
| --- | --- | --- |
| S | 2.0 | The highest possible tier. |
| A | 1.5 |The target displays an exceptional ability within this stat. |
| B | 1.2 |
| C | 1.0 | The middle tier. |
| D | 0.9 | |
| E | 0.7 | |
| F | 0.4 | The lowest possible tier. |
